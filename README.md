# Adaptive Low-Power Congestion-Aware NoC Router for FPGA

## Project Overview

This project presents the design and implementation of an Adaptive Low-Power Congestion-Aware Network-on-Chip (NoC) Router for FPGA. The router dynamically selects routing paths based on congestion information to improve network performance while reducing power consumption.

---

## Objectives

- Design a 5-port NoC Router
- Implement adaptive congestion-aware routing
- Reduce dynamic power consumption
- Verify functionality using Verilog testbenches
- Implement and evaluate on Xilinx FPGA using Vivado

---

## Features

- 5-Port Router
- Circular FIFO Buffers
- Adaptive Routing
- Congestion Monitoring
- Round-Robin Arbitration
- Low-Power Techniques
- FPGA Implementation
