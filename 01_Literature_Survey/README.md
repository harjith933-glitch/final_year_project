
# Literature Survey

This folder contains the literature study conducted for the project:

**Adaptive Low-Power Congestion-Aware NoC Router for FPGA**

The literature survey focuses on Network-on-Chip (NoC) architectures, routing algorithms, congestion-aware routing, adaptive routing, low-power NoC design, and FPGA-based NoC implementations.

## Literature Survey Goals

- Understand existing NoC router architectures.
- Study conventional routing algorithms.
- Study adaptive routing algorithms.
- Analyze congestion-aware routing techniques.
- Study low-power techniques used in NoC routers.
- Identify limitations in existing approaches.
- Identify the research gap.
- Select a suitable base paper.
- Define the proposed project contribution.

## Literature Categories

1. NoC Router Architecture
2. Routing Algorithms
3. Adaptive Routing
4. Congestion-Aware Routing
5. Low-Power NoC Design
6. FPGA-Based NoC Implementation
7. NoC Performance Optimization
