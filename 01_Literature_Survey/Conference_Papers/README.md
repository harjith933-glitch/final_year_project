# Conference Papers

This folder contains conference papers identified during the NoC literature survey.

The papers were selected based on their relevance to the following research areas:

- NoC architecture
- Adaptive routing
- Congestion-aware routing
- Low-power NoC
- FPGA-based NoC
- Router architecture
- Fault-tolerant routing
- Energy-efficient NoC

## Papers Identified in Literature Survey

The current literature survey spreadsheet contains conference papers covering the period:

**2024–2026**

Important research areas identified include:

1. Scalable NoC architectures
2. Routerless NoC
3. Congestion-aware routing

10. Intelligent NoC architectures

## Final Paper Selection

The final set of papers for the project will be selected based on:

- Direct technical relevance to the proposed project.
- IEEE Xplore availability.
- Correct publication year.
- Conference publication status.
- DOI availability where applicable.
- Technical similarity to the proposed router.

## Project-Relevant Literature Themes

The most relevant themes identified from the survey are:

**Adaptive Routing + Congestion Awareness + Low Power + FPGA Implementation**

These four areas form the primary literature foundation for the proposed project.
