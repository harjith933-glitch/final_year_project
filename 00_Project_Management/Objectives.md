# Project Objectives

## Primary Objective

To design and implement an adaptive low-power congestion-aware Network-on-Chip (NoC) router using Verilog HDL and evaluate its performance on FPGA.

## Specific Objectives

1. Design a 5-port NoC router architecture.

2. Design input FIFO buffers for temporary storage of incoming flits.

3. Develop a congestion monitoring mechanism based on buffer utilization.

4. Implement an adaptive routing algorithm that considers congestion conditions.

5. Design an arbitration mechanism to resolve multiple requests for the same output port.

6. Design a crossbar switch for transferring packets between input and output ports.

7. Implement low-power techniques to reduce unnecessary switching activity.

8. Verify individual RTL modules using simulation.

9. Integrate and verify the complete NoC router.

10. Synthesize and implement the design on FPGA using Xilinx Vivado.

11. Analyze FPGA resource utilization, timing, power, latency, and throughput.

12. Compare the proposed design with a conventional routing approach.
